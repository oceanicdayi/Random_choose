# 家事抽籤桶

手機互動式家事抽籤網頁。搖晃手機時，畫面中的抽籤桶與籤支會同步搖晃；之後會抽出一支放大的家事籤，並把抽中的家事項目直接顯示在籤上。

## 線上版本

https://oceanicdayi.github.io/Random_choose/

## 家事項目

目前包含：掃地、拖地、洗衣服、洗廁所、洗牆壁、刷馬桶、折衣服、倒垃圾、擦桌子、洗碗、整理房間、擦窗戶、整理書桌、收拾玩具。

## 使用方式

1. 使用手機開啟線上版本。
2. 按下「啟用手機搖晃」。
3. 允許瀏覽器存取動作與方向感測器。
4. 搖晃手機，等待一支家事籤掉出。
5. 電腦或不支援感測器的裝置可按「模擬搖晃」。

## 技術說明

- 純 HTML、CSS、JavaScript
- 使用 `DeviceMotionEvent` 感應手機搖晃
- 支援 iOS 動作感測權限請求
- 透過 GitHub Actions 自動部署至 GitHub Pages
