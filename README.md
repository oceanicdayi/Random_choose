# 搖搖抽籤桶

手機互動式抽籤網頁。搖晃手機時，畫面中的抽籤桶與籤支會同步搖晃；累積足夠搖晃後，會掉出一支籤並顯示隨機結果。

## 線上版本

https://oceanicdayi.github.io/Random_choose/

## 功能

- 手機加速度與搖晃感測
- 籤桶與多支籤的同步動畫
- 隨機掉出一支籤
- 隨機籤文結果
- 手機震動回饋
- 無感測器時可按「模擬搖晃」測試
- 單一 `index.html`，不依賴外部套件

## 使用

1. 用手機開啟線上版本。
2. 按「啟用手機搖晃」。
3. iPhone／iPad 請允許「動作與方向」權限。
4. 搖晃手機數次，等待籤掉出。

## 修改籤文

編輯 `index.html`，搜尋 `const fortunes`，修改陣列內的文字即可。

## 部署

儲存庫包含 GitHub Pages 工作流程。推送到 `main` 分支後會自動部署。
